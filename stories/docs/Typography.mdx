# Typography

Typography styles used in Hanekokoro UI.

## Font

There's no predefined font choices, nevertheless two choices are provided: Inter and Outfit.

## Header font sizes

There are six heading levels avaliable, mapping the HTML elements `<h1>` to `<h6>`.

* H1: 2.75rem
* H2: 2.5rem
* H3: 2rem
* H4: 1.75rem
* H5: 1.5rem
* H6: 1.25rem

Below demonstrates the actual size with `1rem` = `16px`:

### Inter

<Typeset
  fontSizes={[44, 40, 32, 28, 24, 20]}
  sampleText='Heading example'

/>

## Body fonts

* Large: `1.125rem`
* Base: `1rem`
* Small: `0.875rem`
